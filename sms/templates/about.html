Построение отчета

Сервис предназначен для приема статусов доставки sms-сообещений от провайдера sms-шлюза. Карта сайта представляет собой следующий набор роутов:

/send - предназначен для приема данных от провайдера. Параметры записи передаются через url с валидацией, в случае успеха записываются в БД

/report - предназначен для формирования отчета. Поля формы подставляются в WHERE секцию SQL запроса

/result - предназначен для вывода результата запроса, сформированного в /report роуте, в виде таблицы с возможностью сохранения в xls

/download - служебный роут для скачивания сгенерированного налету отчета
